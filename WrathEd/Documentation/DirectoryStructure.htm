If you are creating a Mod for Tiberium Wars, Red Alert 3, or Tiberium Twilight the name can be no more than 15 characters long.

Required Files:
If you are building a Mod for Tiberium Wars, Red Alert 3, or Tiberium Twilight you have to have a file called mod.xml in your Data folder.
For Kane's Wrath your main file would be called static.xml. This is also the file that is referenced by the default batch files supplied with WrathEd. These names are hard coded and renaming will cause your Mod not to work.

String Table:
Tiberium Wars, Red Alert 3, and Tiberium Twilight support a file called mod.str to store strings. This is a simple text file and only needed if you add or modify text in the game.
An example of an entry in the mod.str file: 
NAME:Orca
"Orca Fighter"
END
Kane's Wrath does not support a mod.str file. Therefor you have to extract and edit the cnc3.csf located in: 
Kane's Wrath\Lang-[language]\1.2\patch2.big
There are several csf editors available but it is recommended to check if it isn't outdated.
