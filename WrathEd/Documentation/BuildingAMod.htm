• Building a Mod
The final step in completing a mod is to build the data for the game. Before you can use a mod, you will need to build the data. Your tool for doing this is BuildMod.bat. BuildMod.bat is a batch file that will gather all of the proper files, run WrathEd, create the .big and .SkuDef files of your mod, and move it to the proper locations.

Using BuildMod.bat
To use BuildMod.bat you will either need to open a command prompt window and navigate to the WrathEd directory, or create a shortcut to it.

• Command Prompt:
To open the command prompt the most easy way is to press WinKey + R, type in cmd and hit enter. From there use cd (Change Directory) to get to the WrathEd directory. Once you are there type in something like the following example command: 
BuildMod.bat SampleMod 1.0
(If you leave name and version blank you are asked to type the name and version.)


• Shortcut:
To create a shortcut right click on the BuildMod.bat and then click on "Create shortcut". Best to name it something like "Build SampleMod". Then right click on it and click Properties. In the new window add something like the following example: 
 SampleMod 1.0
(after the path, outside of the quotation marks)


Launching the Mod in Kane's Wrath
To launch the mod in Kane's Wrath you have to edit the CNC3EP1_[language]_1.2.SkuDef, located in the root of your Kane's Wrath directory. You will need to add a line similar to the following to the top of the file: 
try-add-config C:\Users\[Username]\My Documents\[Game]\mods\SampleMod\SampleMod_1.0.SkuDef
